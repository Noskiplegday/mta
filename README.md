# 🚀 MTA Roleplay Server

Một server **Multi Theft Auto: Roleplay** được build từ đầu, hỗ trợ MySQL, hệ thống nhân vật, và gameplay mở rộng.  
Repo này dành cho những ai muốn học, phát triển hoặc triển khai server RP theo chuẩn hiện đại.

## ⚙️ Cài đặt

1. Clone repo:
   ```bash
   git clone https://github.com/Noskiplegday/mta.git
Copy toàn bộ vào thư mục MTA Server.

Chạy MTA Server.exe.

Kiểm tra file meta.xml và connection.lua để cấu hình database.

✨ Tính năng nổi bật
🔑 Hệ thống đăng nhập MySQL

👤 Quản lý nhân vật (load/save từ DB)

💬 Chat roleplay (/do, /me)

🛠️ Admin tools (teleport, ban, kick)

🚗 Quản lý phương tiện

🏠 Hệ thống nhà cửa, job, tiền tệ (đang phát triển)

📂 Cấu trúc dự án
Mã
server/
 ├── core/           # Script chính
 ├── mysql/          # Kết nối database
 ├── resources/      # Tài nguyên phụ trợ
 ├── vehicle/        # Quản lý xe
 ├── gameplay/       # Logic RP
 └── managers/       # Module quản lý
📝 Hướng phát triển
Thêm hệ thống inventory

Hệ thống economy (tiền, ngân hàng)

Job system (cảnh sát, bác sĩ, taxi…)

Web dashboard quản lý server
